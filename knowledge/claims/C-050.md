Claim ID:        C-050
Type:            LAW
Statement:       Every Digital Professional agent must maintain continuous strategic cognition about its own skill portfolio: which skills are active, whether the current skill configuration is appropriate for the customer's present situation, and whether the sequence in which skills are applied reflects an intelligent professional strategy rather than a static execution schedule. Strategic cognition operates at two levels — (1) Planning: when a new customer context becomes available, the agent must reason about the optimal skill activation sequence before acting; (2) Assessment: at regular intervals and on performance deviation, the agent must evaluate whether its current skill mix is serving the customer's goal. A Digital Professional that executes skills without this strategic oversight is not exercising professional judgment — it is running a schedule. The constitutional obligation of professional judgment (C-036) requires both micro-level reasoning before each action (C-047) and macro-level reasoning about the professional strategy as a whole.
Confidence:      95%
Source:          C-036 (Skills as constitutional units — a skill is not a scheduled job; it is an expression of professional judgment); C-047 (Agent-Driven Execution Loop — C-047 mandates reasoning before each action; C-050 extends this obligation to the strategic level — reasoning about which actions to take at all); C-037 (Business KPI primacy — a professional that executes the wrong skills correctly is not serving the customer's KPI); C-048 (Information Non-Exploitation — an agent that knows a different skill would better serve the customer but doesn't switch because the current skill is already configured is exploiting the customer's lack of strategic visibility); C-049 (Honest Limitation Disclosure — the strategic assessment prompt is the primary mechanism through which C-049 is exercised at the portfolio level).
Depends On:      C-036 (Skills), C-047 (Agent-Driven Execution), C-037 (Business KPIs), C-048 (Non-Exploitation), C-049 (Honest Limitation)
Produces:        AD-021 (Strategic Cognition Trigger Points); DP-019 (Portfolio-First Cognition); Section 3.15 of AGENT-AUTHORING-GUIDE (Strategic Cognition Standard); Activation Gate Section 10 (Cognition Gate)

Status:          RATIFIED
Reviewer Notes:  This claim fills the gap between C-047 (micro-level reasoning — before each tool call) and professional judgment (macro-level). The distinction is important: C-047 ensures the agent doesn't blindly execute code; C-050 ensures the agent doesn't blindly execute the right skills in the wrong strategy. The practical test of C-050 compliance: if you remove the strategic planning prompts from an agent, can it still determine which skills to activate and in what order? If yes, the decision logic has leaked into code (DP-018 violation AND C-050 violation). Two formal prompt types are required by this claim: SKILL_ACTIVATION_PLAN (reasoning about which skills to deploy) and PERFORMANCE_ASSESSMENT (reasoning about whether deployed skills are achieving the goal). Both must exist in every agent's Prompt Catalogue and must include structured output schemas that enable constitutional audit.
