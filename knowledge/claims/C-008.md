Confidence:      100%
Source:          Constitution Article XIV — The Constitutional Chain
Depends On:      NONE
Produces:        NONE (meta-law — governs resolution of all contradictions found during architecture and implementation)

Reviewer Notes:  This is the constitutional operating rule for the entire engineering organization. Its practical meaning: if an ADR contradicts a constitutional article, the ADR must be amended. If an implementation contradicts an ADR, the implementation must change. If runtime evidence contradicts the implementation, it validates the constitution — not the other way around. The Constitutional Chain flows upward for validation and downward for derivation.
